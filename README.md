# smoothfade


## API
```
j

sm.fadeIn({
	'endTime': endTime,
	'startTime': startTime, // optional, default = now
	'targetValue': targetvalue //optional, default = 1
});

sm.fadeOut({
	'endTime': endTime,
	'startTime': startTime, // optional, default = now
	'targetValue': targetvalue //optional, default = 0.00001
});
```
